# AI Language Translator

## Features

- Translate text between multiple languages
- Copy translated text
- Text-to-Speech support
- Input validation
- Simple and responsive interface

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Google Translator API
- gTTS

## Installation

```bash
pip install -r requirements.txt
python app.py
```

Visit:

```
http://127.0.0.1:5000
```
